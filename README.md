# Amazon-Sales-Dashboard

1. Project Overview
This Amazon Sales Dashboard is developed using Power BI to analyze sales performance, order trends, profit, and regional insights. It helps stakeholders understand business performance and customer behavior through interactive visuals.

3. Data Source
Amazon Sales Dataset in CSV format

The dataset contains order date, order status, sales, profit, product category, sub-category, customer segment, payment mode, state, region, and time-based data.

6. Business Insights
The dashboard identifies high-performing categories and customer segments.
It highlights customer payment preferences and order status distribution.
Monthly and yearly sales trends reveal seasonal patterns.
Regional and state-wise analysis supports better decision-making.

7. Key Learnings
Built an end-to-end Amazon sales analytics dashboard.
Strengthened Power BI skills including DAX and data modeling.
Improved ability to convert raw data into business insights.
